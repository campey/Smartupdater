function foo(data) 
{
	$("#example1").html(data.split("").reverse().join(""));
}

If you develop complex application, very often you need to process the same data differently depend on current layout, state, user preferences or other conditions. With Smartupdater you can easily swithch among multiple callback function, which makes your code very modular, flexible and clean.

In this example you can see date in regular and reverse order. It's not necessary to restart smartupdater after it was altered, but if you want to update view immediately, do restart it.
