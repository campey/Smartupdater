$("#example1").smartupdater({
    url : 'php/getsec.php',
    minTimeout: 5000 // 5 seconds
    }, function (data) {
        $("#example1").html(data);
    }
);
	
function suSetTimeout() 
{
    var timeout = $("#newTimeout").val() * 1000;
    $("#example1").smartupdaterSetTimeout(timeout);
}

At any time you can set any timeout which is right for current task your application is executing. See how it works in this demo. (Set interval in seconds, not miliseconds)
