Another case: depend on current layout, state, user preferences or other conditions you want to request different data. For example, user iconized module: you don't need request regular data any more, but want to poll alerts or notification only. With Smartupdater you can easily swithch among multiple URLs, which makes your code very modular, flexible and clean.

In this example you can poll eather regular getsec.php or funny reverse date layout. It's not necessary to restart smartupdater after it was altered, but if you want to update view immediately, do restart it.
